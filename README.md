# POPMART Renewal

트렌디하고 키치한 무드로 재해석한 팝마트 리뉴얼 팀 프로젝트입니다.  
제품 소개부터 인터랙션, 애니메이션 등 시각적 요소에 집중하여 브랜드의 개성을 표현했습니다.

<br/>

## 💡 기획 포인트

- **키치하고 감각적인 비주얼 구성**
- **GSAP, Swiper 등 다양한 라이브러리 활용**
- **Hover / Scroll 기반 인터랙션 강화**
